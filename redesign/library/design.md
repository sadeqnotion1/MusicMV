# 📂 Screen 2: Local Library & Catalogs (Widescreen Desktop Lock)

This section lets the user browse and audit their locally sorted artist folders, including circular icons and Top 50 missing hits.

---

## 💻 Master Google Stitch Prompt for Library Redesign

Copy and paste this exact prompt into **[stitch.withgoogle.com](https://stitch.withgoogle.com/)** to generate the Local Library dashboard:

```text
Design a ultra-premium, modern, responsive widescreen desktop dashboard named 'MusicMV Library Explorer' (Viewport 1920x1080).
Aesthetic: Dark cyber-obsidian with glassmorphic elements.
Colors: Base background (#05060A), card background (rgba(13, 17, 28, 0.7)), glowing borders (rgba(139, 92, 246, 0.15)), highlights (#06B6D4) and (#8B5CF6).

Layout Grid (Left-to-Right):
1. Left Widescreen Sidebar (260px): brand title, SVG icons for navigation ('Intel Search', 'Library Explorer', 'Console Panel'). No emojis.
2. Center Main Panel:
   - Stats row: 3 horizontal stats boxes displaying: 'Sorted Artists: 142', 'Total Videos: 1,840', and 'Missing Hits: 56' (using cyber-cyan and electric violet outlines).
   - Artists Grid: A 6-column widescreen grid displaying circular artist cards. Each card contains a circular avatar image with a glowing border outline, artist name, and a badge with the count of local files (e.g., '12 Videos').
3. Right Side Drawer (350px): 'Missing Tracks Auditor' comparing local folders against streaming charts. Includes a list of missing tracks with song title, estimated file size, and a 'Get ↓' action button next to each missing hit.
```
